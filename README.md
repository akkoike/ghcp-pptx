# GHCP-PPTX

GitHub Copilot を使って、プロンプトファイルを実行するだけでパワーポイント資料を自動作成するツールです。

## 使い方

### 1. プロンプトファイルを編集する

`.github/prompts/pptx-create-env.md` を開き、`# 指示` セクションに作成したいパワーポイントのテーマを記載します。

```markdown
# 指示 <!-- 以下の1行を作成したい資料のテーマに合わせて修正してください !-->
- Azure Firewall の詳細仕様をパワーポイントでまとめてください。
```

### 2. プロンプトファイルを実行する

VS Code のチャットパネルを開き、`pptx-create-env` プロンプトを実行します。

1. チャットパネルで `/` を入力し、プロンプト一覧から `pptx-create-env` を選択します
2. 実行すると、以下の 4 つのエージェントが自動的に順番に実行されます

| 順序 | エージェント | 処理内容 |
|------|-------------|----------|
| 1 | makemd | テーマに基づきスライド構成のマークダウンファイルを作成します |
| 2 | checkmd | 作成されたマークダウンの事実確認・品質レビューを行います |
| 3 | createpptx | レビュー済みマークダウンからパワーポイントファイルを生成します |
| 4 | finalize | テンプレート適用・フォント統一・仕様チェックなどの最終調整を行います |

### 3. 完成ファイルを確認する

処理が完了すると、テーマ名のディレクトリ配下にファイルが生成されます。基本定義上の想定構成は以下です。

```
<テーマ名>/
├── docs/                           # 完成した .pptx ファイル
│   └── <テーマ名>.pptx
├── js/                             # pptxgenjs を使用したスライド生成スクリプト
│   └── create-slide.js
├── py/                             # python-pptx を使用した生成スクリプト
│   └── create-slide.py
└── md/                             # スライド構成マークダウン
    ├── <テーマ名>.md               # スライド構成定義
    └── <テーマ名>-Checked.md       # レビュー結果
```

`<テーマ名>/docs/<テーマ名>.pptx` が完成したパワーポイントファイルです。

## 前提条件

- VS Code に GitHub Copilot 拡張機能がインストールされていること
- Node.js がインストールされていること
- PptxGenJS を利用できる環境があること
- Python がインストールされていること
- `python-pptx` パッケージがインストールされていること（`pip install python-pptx`）

## フォルダ構成

```
GHCP-PPTX/
├── .github/
│   ├── agents/                     # 各処理ステップのエージェント定義
│   │   ├── makemd.md               # マークダウン作成エージェント
│   │   ├── checkmd.md              # マークダウンレビューエージェント
│   │   ├── createpptx.md           # パワーポイント作成エージェント
│   │   └── finalize.md             # 最終調整エージェント
│   ├── instructions/               # 基本仕様定義
│   │   └── createpptx.instructions.md
│   ├── prompts/                    # プロンプトファイル
│   │   └── pptx-create-env.md      # ★ このファイルを編集して実行
│   └── skills/                     # スキル定義
│       └── create-pptx/
│           └── SKILL.md
├── templates/                      # 表紙・フッターのテンプレート
└── README.md
```
